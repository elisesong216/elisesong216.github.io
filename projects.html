neural networks

In highschool, I programmed a neural network that can identify handwritten digits after training on the MNIST digits dataset. Accompanied by a graphics tool, the user can draw a digit and the neural network will identify the number 0-9.

enigma

I programmed Enigma for a CS 3110: Data Structures and Functional Programming assignment. I implemented a key substitution cipher for each of Enigma's mechanisms, including rotors, reflectors, and the plugboard.
